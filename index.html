Beacon Beam Creator

Find how to create the color you want for a Beacon Beam with stained glass.

  • Beacon to Color mode takes a desired color and attempts to find how to make that color.
  • Glass Combination to Color mode takes a list of glass and gives the color made by the beacon.
  • Color to Color mode attempts to find the glass needed to go from one color to another color.
